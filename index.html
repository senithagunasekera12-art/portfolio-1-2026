01. Data

Each block contains a payload—transactions, smart contract state changes, or oracle data. This payload is serialized into a byte string.

02. Hash

SHA-256 digests the data into a 64-character hexadecimal fingerprint. Alter one bit, the entire hash changes unpredictably.

03. Link

Each block embeds the previous block's hash. To tamper with history, you must re-mine every subsequent block faster than the network.
